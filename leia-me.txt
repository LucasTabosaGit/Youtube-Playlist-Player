Abra o Terminal na Pasta do Projeto:

Navegue até a pasta onde você salvou os arquivos do player. Você pode fazer isso da seguinte forma:
Abra o explorador de arquivos.
Na barra de endereço, digite cmd e pressione Enter. Isso abrirá o Prompt de Comando na pasta atual.
Instale as Dependências:

No terminal que abriu na pasta do projeto, digite o seguinte comando e pressione Enter:

npm install

Este comando instalará todas as dependências necessárias para o player. Certifique-se de estar conectado à internet, pois o npm baixará os pacotes necessários.
Inicie o Servidor do Player:

Após a conclusão da instalação das dependências, mantenha o terminal aberto.
Procure pelo arquivo chamado player youtube.bat na pasta. Este arquivo será responsável por iniciar o servidor local.
Mantenha o Terminal Aberto:

É importante manter o Prompt de Comando aberto enquanto o servidor está em execução. Isso garantirá que o player funcione corretamente e que você possa visualizar qualquer saída ou mensagem relevante.

Você pode criar um atalho para o desktop do arquivo .bat assim será mais fácil você acessar quando quiser, também é possível personalizar o ícone do atalho nas propriedades.
